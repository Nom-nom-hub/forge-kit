# Security Policy

Thanks for helping keep Forge Kit safe for everyone.

## Reporting Security Issues

If you believe you have found a security vulnerability, please report it to us through coordinated disclosure.

**Please do not report security vulnerabilities through public GitHub issues, discussions, or pull requests.**

Instead, please send an email to opensource-security@example.com.

Please include as much of the information listed below as you can:
- The type of issue
- Full paths of source file(s) related to the manifestation of the issue
- Step-by-step instructions to reproduce the issue
- Impact of the issue
