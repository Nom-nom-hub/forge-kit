# FORGE Slash Commands Reference

## Core Commands

| Command | Agent Skill | Description |
|---------|-------------|-------------|
| `/forge.signal` | `forge-signal` | Capture a new signal |
| `/forge.hypothesize` | `forge-hypothesize` | Form a hypothesis from signals |
| `/forge.decide` | `forge-decide` | Record a decision |
| `/forge.feature` | `forge-feature` | Create a feature from a hypothesis |
| `/forge.experiment` | `forge-experiment` | Design and run an experiment |
| `/forge.release` | `forge-release` | Create a release manifest |
| `/forge.retrospect` | `forge-retrospect` | Generate retrospective signals |
