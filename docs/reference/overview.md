# Forge CLI Reference

## Commands

| Command | Description |
|---------|-------------|
| `forge init` | Initialize a new FORGE project |
| `forge version` | Show version |

## Artifact Directories

| Directory | Purpose |
|-----------|---------|
| `.forge/signals/` | Signal artifacts (raw observations) |
| `.forge/hypotheses/` | Hypothesis artifacts (testable beliefs) |
| `.forge/decisions/` | Decision artifacts (recorded choices) |
| `.forge/features/` | Feature artifacts (delivery containers) |
| `.forge/experiments/` | Experiment artifacts (time-boxed tests) |
| `.forge/releases/` | Release manifests |
| `.forge/retrospectives/` | Retrospective signals |
| `.forge/cognitive-contracts/` | Cognitive contracts |
| `.forge/graph/` | Graph edge index (auto-generated) |
